# Teknik Altyapı ve SEO: Kurulum ve Görünürlük Stratejisi

Bu doküman, web sitesinin teknoloji yığınını, güvenlik standartlarını ve arama motoru optimizasyonu (SEO) planını tanımlar.

## 1. İçerik Yönetim Sistemi ve Framework

*   **Platform:** Next.js (Yüksek performans, SEO uyumluluğu ve modern teknoloji yığını nedeniyle).
*   **Stil Kütüphanesi:** Tailwind CSS (Hızlı ve özelleştirilebilir tasarım için).
*   **UI Bileşenleri:** HeroUI, Radix UI veya benzeri modern bileşen kütüphaneleri.

## 2. Tavsiye Edilen Teknolojiler ve Bağımlılıklar

| Fonksiyon | Tavsiye Edilen Çözüm |
| :--- | :--- |
| **SEO** | next-seo veya Metadata API |
| **Güvenlik** | NextAuth.js / Middleware |
| **İçerik Yönetimi** | Strapi, Contentful veya Markdown tabanlı yönetim |
| **Formlar** | React Hook Form + Zod |
| **Erişilebilirlik** | Radix UI Primitives |
| **Dağıtım** | Railway / Vercel |

## 3. Alan Adı ve Hosting (Barındırma)

*   **Alan Adı:** `tkbbursa.org` veya `turkkadinlarbirligibursa.org` (Öncelik .org).
*   **Hosting Kriterleri:**
    *   Türkiye lokasyonlu veri merkezi.
    *   Ücretsiz SSL Sertifikası (HTTPS).
    *   Litespeed sunucu teknolojisi.
    *   Günlük yedekleme.

## 4. SEO (Arama Motoru Optimizasyonu)

### 4.1. Anahtar Kelime Stratejisi
*   **Marka:** TKB Bursa, Türk Kadınlar Birliği Bursa Şubesi, Tijen Sözeri.
*   **Hizmet:** Bursa eğitim bursu, Bursa kadın dernekleri, kadın hakları savunuculuğu.
*   **Konum:** Bursa kadın etkinlikleri, Merinos AKKM etkinlikleri.

### 4.2. Teknik SEO Adımları
*   Tüm görseller için "alt text" (alternatif metin) girişi.
*   Sayfa yükleme hızının optimizasyonu (LCP < 2.5s).
*   XML Site Haritası oluşturulması ve Google Search Console kaydı.
*   Her sayfa için özgün meta başlık ve açıklamaların yazılması.
